TeleFormation
「VRの会話の輪に、もっと自然に、もっと心地よく。」
TeleFormation は、F-Formation（フォーメーション）理論を用いることで、共有仮想空間（Social VR）におけるグループ会話への参加をより快適にするための可視化インターフェースです。

概要
本プロジェクトは、Social VRにおいて既存の会話グループへ新規ユーザーが合流する際の心理的ハードルを下げ、快適なインタラクションを実現することを目的としています。エドワード・T・ホールの近接学（Proxemics）に基づき、ユーザー間の適切な距離感や空間の「空気」を可視化することで、スムーズな合流を支援します。

特徴
F-Formationの活用: 会話グループが形成する空間的なパターンを解析・可視化します。

快適性の向上: ユーザーが会話に参加する際の心理的な負担を軽減し、自然な合流を促します。

VR特化型UI: Social VR環境での利用に最適化された直感的なインターフェースです。

開発環境
Game Engine: Unity 6.1

Target Hardware: Pico 4 Ultra

Assets: Mixamo (Virtual Agents)

Data Analysis: Python (pandas, matplotlib)
